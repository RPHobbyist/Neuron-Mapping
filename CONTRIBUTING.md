# Contributing to Neuron Mapping

Thank you for your interest in contributing to Neuron Mapping! We welcome contributions from the community to help make this project better.

## Getting Started

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your local machine.
3.  **Install dependencies** using `npm install`.

## Development Workflow

1.  Create a new branch for your feature or bug fix: `git checkout -b feature/my-new-feature`.
2.  Make your changes and commit them with descriptive commit messages.
3.  Running in development mode: `npm run dev`
4.  Ensure your code follows the project's coding style and conventions.

## Pull Requests

1.  Push your branch to your forked repository.
2.  Open a Pull Request (PR) against the `main` branch of the original repository.
3.  Provide a clear description of your changes and why they are necessary.
4.  Address any feedback or review comments.

## Reporting Issues

If you encounter any bugs or would like to request a new feature, please open an issue on the GitHub repository. Please provide as much detail as possible, including steps to reproduce the issue.

## Code of Conduct

Please adhere to our Code of Conduct in all interactions within the community.

Thank you for contributing!
