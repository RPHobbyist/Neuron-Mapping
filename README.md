A visual mind mapping tool for organizing ideas and brainstorming.

## Features

- Create beautiful mind maps
- Multiple template types
- Save and load maps
- Color-coded nodes
- Drag and drop interface

## Getting Started

```bash
npm install
npm run dev
```
